بناء وتحميل الإضافة في متصفح Chrome

- لبناء حزمة الإضافة إلى مجلد `dist/` شغّل:

```
npm run build:chrome
```

- بعد الانتهاء، حمّل الإضافة في Chrome (أو Edge) عبر: الإعدادات → الملحقات → وضع المطور → تحميل مجلد غير مضغوط → اختر مجلد `dist/`.

- ملاحظة: هذه الإضافة تستخدم ملف `index.html` الناتج كـ popup. إذا كنت تحتاج إلى أي إعدادات إضافية (أيقونات، أذونات، خدمة عاملة)، أخبرني لأضيفها.

نشر أوتوماتيكي عبر GitHub → Vercel

- لإعداد نشر تلقائي إلى Vercel باستخدام GitHub Actions:
	- أنشئ سرّ GitHub باسم `VERCEL_TOKEN` يحتوي على توكن Vercel الخاص بك.
	- عند الضغط إلى الفرع `main` سيقوم الـ workflow ببناء المشروع ونشره إلى Vercel.

- ملاحظات:
	- لتوليد `VERCEL_TOKEN` سجّل دخولك إلى https://vercel.com → Settings → Tokens.
	- يمكنك تعديل مسار البناء أو أوامر البناء في [package.json](package.json).

- ملف إعدادات Vercel

	- أضفت ملف `vercel.json` في جذر المستودع لتحديد مجلد الإخراج (`dist`) وتوجيه جميع المسارات إلى `index.html` لتطبيق صفحة واحدة.
	- إن أردت تضمين Project/Org IDs يمكنك إضافتها إلى `vercel.json` أو ضبطها عبر واجهة Vercel، لكن لا تشارك التوكنات علنًا — استخدم `VERCEL_TOKEN` في أسرار GitHub.
